# WeChat企微通知模板

## 概述

本文档描述了Bantu签证助手中WeChat企微通知的三个阶段模板。系统会根据签证剩余天数自动选择相应的通知模板。

## 通知阶段

### 阶段一：5天内过期（正常提醒）

**触发条件**: 签证剩余天数 ≤ 5天

**通知标题**: 📋 签证即将过期提醒 - 5天内

**通知颜色**: 橙色 (#FFA500)

**通知内容示例**:
```
📋 签证即将过期提醒 - 5天内

以下签证将在5天内过期，请及时处理

---

1. 张三
- 护照号: `A12345678`
- 签证类型: B211A
- 到期日期: 2026-03-20
- 剩余天数: 5天

2. 李四
- 护照号: `B87654321`
- 签证类型: C31x
- 到期日期: 2026-03-21
- 剩余天数: 4天

---

⏰ 发送时间: 2026-03-15 09:00:00
```

**处理建议**:
- 联系客户确认签证状态
- 准备续签所需文件
- 安排续签时间

---

### 阶段二：3天内过期（紧急提醒）

**触发条件**: 签证剩余天数 ≤ 3天

**通知标题**: 🔴 签证紧急提醒 - 3天内

**通知颜色**: 红色 (#FF6B6B)

**通知内容示例**:

---

## 通知发送方式

### 自动发送（每天9:00 AM）

系统会在每天早上9:00 AM自动检查所有启用提醒的签证，并根据剩余天数分别发送三个阶段的通知。

### 手动发送

在Reminders页面点击"企微通知"按钮，可以立即发送当前的所有阶段通知。

## 配置说明

### 环境变量

```env
# WeChat企微Webhook URL
WECHAT_WEBHOOK_URL=https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=YOUR_WEBHOOK_KEY
```

### 获取Webhook URL

1. 登录企业微信管理后台
2. 进入"应用与集成" → "应用" → "机器人"
3. 创建新的机器人或选择现有机器人
4. 复制Webhook URL

---

## 消息格式

所有通知使用Markdown格式发送，支持以下格式：

- **粗体**: `**文本**`
- *斜体*: `*文本*`
- `代码`: `` `文本` ``
- 标题: `# 标题`
- 列表: `- 项目`

---

## 故障排查

### 通知未发送

1. 检查`WECHAT_WEBHOOK_URL`是否正确配置
2. 检查Webhook URL是否过期
3. 查看服务器日志中的错误信息

### 通知格式错误

1. 确保Webhook URL指向正确的企业微信机器人
2. 检查网络连接
3. 验证Markdown格式是否正确
